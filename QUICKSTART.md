# Quick Start - CSS Scanner Pro

## Install (2 minutes)

1. Open Chrome
2. Go to `chrome://extensions/`
3. Turn ON "Developer mode" (top right)
4. Click "Load unpacked"
5. Select this folder

## Use (30 seconds)

1. **Open any website**
2. **Click the extension icon** or press `Ctrl+Shift+S`
3. **Hover** over any element
4. **Click** to copy CSS
5. **Space** to pin
6. **Esc** to close
