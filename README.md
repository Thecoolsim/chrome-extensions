# Chrome Extensions

This repository contains my Chrome extensions.

## Extensions

### [CSS Scanner Pro](css-scanner-pro/)

The fastest and easiest way to inspect, copy and analyze CSS styles on any webpage.

**Features:**
- Instant CSS inspection on hover
- Multi-tab interface (CSS, HTML, Source, Editor)
- Live CSS editor with real-time preview
- CodePen export
- Syntax highlighting
- Multi-language support (EN, FR, ES, DE)
- Keyboard shortcuts
