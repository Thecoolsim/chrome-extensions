## How to Submit Localized Listings

1. Go to Chrome Web Store Developer Dashboard
2. Navigate to your extension's listing
3. Click "Store Listing" tab
4. At the top, you'll see "Add a new language"
5. Select each language (fr, es, de) and paste the content above
6. Save each language listing
7. Submit for review

The Chrome Web Store will automatically show the appropriate language based on the user's browser settings.
