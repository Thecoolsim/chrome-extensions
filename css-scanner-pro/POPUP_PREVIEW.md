## Animations

### Fade-In (on open)
```
0ms:   opacity: 0
300ms: opacity: 1
```

### Pulse (status dot)
```
Green dot cycles:
0s:   opacity 100%
1s:   opacity 50%
2s:   opacity 100%
(repeats infinitely)
```

### Hover Transitions
```
All hover effects: 200ms ease
- Background color
- Transform (slide/lift)
- Shadow
```
