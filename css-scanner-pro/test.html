How to Use

  1. Click the CSS Scanner Pro extension icon in your toolbar
  2. Or press Ctrl+Shift+S (Cmd+Shift+S on Mac)
  3. Hover over any element to see its CSS
  4. Click to copy CSS to clipboard
  5. Press Space to pin CSS blocks
  6. Press Escape to close
