CSS Scanner Pro

Quick Guide & Shortcuts

Extension Ready
Quick Start
Activate Scanner Ctrl+Shift+S
Toggle Grid Overlay Ctrl+Shift+G
Scan Parent Element Ctrl+Shift+E
How to Use
1
Press Ctrl+Shift+S or click the extension icon to activate
2
Hover over any element on the page to see its CSS
3
Use the inspector tabs: CSS, HTML, Source, Editor
4
Click Copy to copy code to clipboard
5
Press Space to freeze/unfreeze, Esc to close
