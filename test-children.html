  1. Activate CSS Scanner (click icon or press Ctrl+Shift+S)
  2. Hover over a parent element with children (like the cards below)
  3. Look at the CSS tab in the inspector
  4. Find the checkbox: "Include child elements CSS"
  5. Check the box to include CSS for all nested children
  6. Uncheck the box to see only the parent's CSS
  7. Compare the difference!
